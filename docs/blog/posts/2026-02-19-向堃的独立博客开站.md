# 向堃的独立博客开站---写在人生40
## 
人生40，开一个独立博客，放在 GitHub 上。不追热点，只写沉淀：读书观影、技术折腾、生活复盘。
我相信：没有完美的系统，所有的系统都需要闭环改善；每个人都有责任尽力让它更好。我们不是只为自己而活，更可能是为了未来而活。
借《镖人》的江湖提醒自己——道义不在口号里，在选择与承担里。欢迎来坐。
