# 命王渤的命运
## 
“高智商/高才华”不等于“高心智”。
如果一个人只有“才华”这种单一资质，而没有在平时通过小挫折去刻意练习**“主客体剥离”，没有建立起“斯多葛式的防御机制”**，那么当命运突然发难时，他越聪明，反而越容易向内攻击自己，最终走向崩溃。
